This folder is for writing all pages of the application.

Use a suitable folder structure for organizing your pages. For example:

- If this is a user login page, use: /pages/user/login.html
- If this is a user create account page, use: /pages/user/createAccount.html

Organize other pages similarly, grouping them by feature or section for better maintainability.

If you are writing a general page that applies to all users (not specific to a feature or section), create the file directly in the /pages directory. For example:

- /pages/about.html
- /pages/contact.html

This helps keep general-purpose pages easily accessible and separate from feature-specific folders.
