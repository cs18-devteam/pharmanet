# Pharmanet

## Getting Started

Follow these steps to set up and run the project:

1. **Install dependencies**  
    ```bash
    npm install
    ```

2. **Start the development server**  
    ```bash
    npm run dev
    ```
    `After step 2 you ready to go`

3. **(Optional) Start the development API server**  
    ```bash
    npm run dev-server
    ```
